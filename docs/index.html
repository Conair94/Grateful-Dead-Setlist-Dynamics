About: This interactive visualization explores the setlist dynamics of the Grateful Dead. It maps how songs transition into one another, allowing you to trace historical patterns, identify common pairings, and even simulate realistically generated setlists based on different eras of the band's 30-year history.

Timeframe Filter: Adjust the dates to recalculate connections based on that specific era.

Graph Mode: Raw groups all plays into one node. Detailed creates separate nodes for Set 1, Set 2, and Epilogue.

Dynamic Coloring: By default, Red nodes indicate Set 1 heavy songs; Blue nodes indicate Set 2+ heavy songs. In Detailed Mode, coloring represents how early (red) or late (blue) in the set a song is played.

Generative Walk: Simulates a setlist based on historical probabilities from your selected timeframe.

Interactivity: Drag nodes to rearrange them. Click a node to isolate its historical transitions.
